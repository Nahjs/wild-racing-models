# wild-racing-models

这个仓库包含狂野赛车游戏的3D模型文件。

## 模型说明

该仓库包含多个汽车品牌的3D模型，包括：
- McLaren
- Lamborghini
- Ferrari
- Bugatti
- Aston Martin
- Koenigsegg
- Czinger
... ...

## Usage

Access models via jsDelivr CDN:

```
https://cdn.jsdelivr.net/gh/Nahjs/wild-racing-models@master/models/[brand]/[model-name].glb
```
