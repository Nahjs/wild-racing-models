# Wild Racing Models

This repository contains 3D models for the Wild Racing game.

## Usage

Access models via jsDelivr CDN:

```
https://cdn.jsdelivr.net/gh/Nahjs/wild-racing-models@master/models/lamborghini/[model-name].glb
```

## Models List
